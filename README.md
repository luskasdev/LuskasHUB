## 💎 Design & Experiência

O projeto foi construído focando em **User Experience (UX)** moderna, utilizando:

- 🌑 **Dark Mode Nativo**: Interface otimizada para reduzir a fadiga ocular.
- ✨ **Efeito Hover 3D**: Cartões interativos que reagem ao movimento do mouse.
- 📱 **Totalmente Responsivo**: Layout que se adapta perfeitamente a qualquer dispositivo.
- 🚀 **Performance**: Código limpo em HTML5 e CSS3 puro, sem dependências pesadas.
